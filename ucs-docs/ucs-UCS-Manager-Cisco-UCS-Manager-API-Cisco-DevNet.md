## Getting Started with UCS Manager

Get started by understanding the fundamentals for the working with UCS Technology. UCS Manager is the single point of management for a UCS Domain.

UCS Domains can contain up to twenty blade chassis with each chassis holding up to eight half-width blades. UCS Domains can also manage UCS Rack-Mount servers. In total UCS Manager can manage up to one hundred and sixty servers.

[Get More Information](https://www.cisco.com/c/en/us/products/servers-unified-computing/index.html)

## UCS Manager Python SDK

Cisco UCS Manager Python SDK for UCS Manager is a Python module that helps automate and manage configurations within UCS Manager including service profiles, policies, pools, equipment, network and storage.

Cisco UCS Manager Python SDK enables easy integration with existing IT management processes and tools. The Python SDK manipulates the Cisco UCS Management Information Tree (MIT). The Python SDK allows you to query, create, modify, and delete the Managed Objects (MOs) in the tree.

[Get More Information](https://communities.cisco.com/docs/DOC-64378)

## UCS Manager PowerTool

Cisco UCS PowerTool for UCS Manager is a PowerShell module that helps automate and manage configurations within UCS Manager including service profiles, policies, pools, equipment, network and storage.

Cisco UCS PowerTool enables easy integration with existing IT management processes and tools. The PowerTool Cmdlets manipulate the Cisco UCS Management Information Tree (MIT). The PowerTool Cmdlets allow you to query, create, modify, and delete the Managed Objects (MOs) in the tree.

[Get More Information](https://communities.cisco.com/docs/DOC-37154)
