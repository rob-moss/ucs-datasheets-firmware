## Search Guide for Other Prompts

When `query` or `query-mcp` prompts need to find documentation:

1. **If searching for CLI commands:** Filter by `type:CLI` + relevant scope (FabricInterconnect, Server, etc.)
   - Example: `type:CLI` + `scope:FabricInterconnect` → UCS Manager CLI References

2. **If searching for API documentation:** Filter by `type:API` + desired mode
   - Example: `type:API` + `mode:Intersight` → Intersight REST API

3. **If looking for firmware/version compatibility:** Use `scope:Firmware` + specific version mode
   - Example: `scope:Firmware` + `mode:IMM` → Firmware Matrices

4. **If need configuration guidance:** Filter by `type:Guide` + `feature:Configuration`
   - Example: `type:Guide` + `feature:Configuration` → Administration guides

5. **If troubleshooting issues:** Filter by `content:Troubleshooting` or `type:Troubleshooting`
   - Example: `content:Troubleshooting` → Troubleshooting guides

6. **If looking for architecture/design guidance:** Use `type:Architecture` + `content:BestPractices`
   - Example: `type:Architecture` + `content:BestPractices` → Design guides

7. **If researching version-specific changes:** Use `type:Release-Notes` + desired version
   - Example: `type:Release-Notes` + `mode:IMM` → IMM Release Notes

8. **If need hardware specifications:** Filter by `type:Reference` + `scope:` (Server, FabricInterconnect, etc.)
   - Example: `type:Reference` + `scope:FabricInterconnect` → FI Datasheets
