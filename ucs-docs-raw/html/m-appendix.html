Appendix

This section provides a list of terminologies that are used in this document.

Name Description

Direct-Connect

C-Series VIC connections that are plugged directly into the Fabric Interconnect port.

Ethernet Port

A generic term for the opening on the side of any Ethernet node, typically in an Ethernet NIC or LAN switch, into which an Ethernet cable can be connected.

Fabric Port Channel

Fibre Channel uplinks defined in a Cisco UCS Fabric Interconnect, bundled together and configured as a port channel, allowing increased bandwidth and redundancy.

FCoE

Fibre Channel over Ethernet. A computer network technology that encapsulates Fibre Channel frames over Ethernet networks.

KVM

Keyboard, video, and mouse

MAC address

A standardized data link layer address that is required for every device that connects to a Logical Area Network (LAN).

Port Mappings

Identifies the ports that are used for specific cable connections between the Fabric Interconnect and other devices.
